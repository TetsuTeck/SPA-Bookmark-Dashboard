# Bookmark Dashboard 🌐

Markdown（マークダウン）を記述するだけで、自分専用の美しく機能的なWebブックマークポータルを自動生成する、1ファイル完結型のダッシュボードツールです。
データベースサーバーを介さず、登録したリンクや各種お気に入り設定はすべてあなたのブラウザ（LocalStorage）の中に安全に自動保存されます。

---

## ✨ 主な特徴 (Features)

* **1ファイル完結 (Single Page Application)**: サーバー不要、HTMLファイルをブラウザで開くだけで爆速動作。
* **2段式超ワイド検索バー**: 画面幅100%をフル活用し、タイトル・URL・備考をリアルタイムに横断検索。
* **ハイブリッド・マルチレイアウト**: 詳細な情報を一覧できる「テーブル表示」と、最大10列まで可変可能な「タイル表示」を1タップ切り替え。
* **個別画像＆フォールバック対応**: タイルごとに任意の画像（Web URL / ローカル相対パス）をセット可能。リンク切れ時も標準アイコンへ安全に自動フォールバック。
* **動的ソート機能**: 「記述順」「名前順（A-Z）」「クリック回数順」の3つのソートモードを搭載。
* **160種超アイコンピッカー**: カタログからアイコンを選択するだけで、裏側のマークダウンコードを自動書き換え。
* **Canvas背景圧縮アップロード**: お気に入りの画像を背景に設定可能（ブラウザ制限を回避する自動軽量化付き）。
* **ローカルファースト＆セキュア**: 外部へのデータ送信は一切なし。完全プライベートな個人専用仕様。

---

## 🚀 使い方 (How to Use)

### 1. 起動方法 (Installation & Run)
1. 本リポジトリから `index.html` をダウンロード（またはコードをコピーしてローカルに保存）します。
2. 保存した `index.html` を、普段お使いのWebブラウザ（Google Chrome, Edge, Safari等）にドラッグ＆ドロップして開きます。

### 2. ブックマークの編集・登録 (Editing Bookmarks)
1. 画面右上にある **「エディタを開く」** ボタンをクリックします。
2. 画面全体に広がるエディタ領域に、以下の文法ルールに従ってブックマークを自由に記述します。
3. 記述が終わったら **「編集を終了してダッシュボードを見る」** をクリックすれば、あなた専用のタブとカードが即座に生成されます。

```markdown
# 🌐 カテゴリ名（シャープの後ろにスペースを入れてカテゴリを定義）
