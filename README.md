# Bookmark Dashboard 🌐

Markdown（マークダウン）を記述するだけで、自分専用の美しく機能的なWebブックマークポータルを自動生成する、1ファイル完結型のダッシュボードツールです。
データベースサーバーを介さず、登録したリンクや各種お気に入り設定はすべてあなたのブラウザ（LocalStorage）の中に安全に自動保存されます。

---

## ✨ 主な特徴 (Features)

* **1ファイル完結 (Single Page Application)**: サーバー不要、HTMLファイルをブラウザで開くだけで爆速動作。
* **ローカル自動スクリーンショット保存 (新機能)**: 外部のクラウドサービスに依存することなく、ローカルのヘッドレスChromeを利用してリンク先スクリーンショットを自動生成し、フォルダに保存・キャッシュ可能。
* **2段式超ワイド検索バー**: 画面幅100%をフル活用し、タイトル・URL・備考をリアルタイムに横断検索。
* **ハイブリッド・マルチレイアウト**: 詳細な情報を一覧できる「テーブル表示」と、最大10列まで可変可能な「タイル表示」を1タップ切り替え。
* **個別画像＆フォールバック対応**: タイルごとに任意の画像（Web URL / ローカル相対パス）をセット可能。画像読み込みエラー時も標準アイコンへ安全に自動フォールバック。
* **動的ソート機能**: 「記述順」「名前順（A-Z）」「クリック回数順」の3つのソートモードを搭載。
* **160種超アイコンピッカー**: カタログからアイコンを選択するだけで、裏側のマークダウンコードを自動書き換え。
* **Canvas背景圧縮アップロード**: お気に入りの画像を背景に設定可能（ブラウザ制限を回避する自動軽量化付き）。
* **ローカルファースト＆セキュア**: 外部へのデータ送信は一切なし。完全プライベートな個人専用仕様。

## 📸 自動スクリーンショット取得機能 (Auto-Screenshot)

画像が設定されていないブックマークについて、リンク先ホームページのスクリーンショットをローカル環境で自動取得し、キャッシュ画像としてタイル表示できます。

### 特徴（完全無料・無制限・高プライバシー）
あなたのPCのローカル環境に画像を保存・キャッシュします。外部APIの制限を一切受けることなく、プライバシーを守りながら無制限にスクリーンショットを取得できます。

* **必要な準備**: PCに Node.js と Google Chrome がインストールされていること。
* **起動手順**:
  1. ターミナル（コマンドプロンプトやPowerShell等）で本ディレクトリを開き、以下を実行してローカルサーバーを起動します：
     ```bash
     node server.js
     ```
  2. ブラウザで `http://localhost:3000` を開きます。
  3. ダッシュボードのヘッダーにある「スクショ自動取得」をオンにします。
  4. 初回表示時に自動で `screenshots/` フォルダが作成され、キャプチャ画像が保存されます。
  5. **手動リフレッシュ機能**: スクショ自動取得がオフの状態でも、画像エリアにカーソルを合わせると表示される左上の「更新」マーク（またはリストビューの更新ボタン）をクリックすれば、いつでも個別に最新のキャプチャを撮影・表示できます。

## 🚀 使い方 (How to Use)

### 1. 起動方法 (Launch)
* **ローカルサーバー経由（スクリーンショット機能対応・推奨）**:
  1. ターミナルで `node server.js` を実行し、ブラウザで `http://localhost:3000` を開きます。
* **スタンドアロン（静的ファイル）**:
  1. 本リポジトリの `spa_bookmark_dashboard.html` をダウンロードし、ダブルクリックしてブラウザで直接開きます。（※この場合、スクリーンショット機能は利用できません）

### 2. ブックマークの編集・登録 (Editing Bookmarks)
1. 画面右上にある **「エディタを開く」** ボタンをクリックします。
2. 画面全体に広がるエディタ領域に、以下の文法ルールに従ってブックマークを自由に記述します。
3. 記述が終わったら **「編集を終了してダッシュボードを見る」** をクリックすれば、あなた専用のタブとカードが即座に生成されます。

#### 📝 マークダウンの記述ルール
```markdown
# 🌐 カテゴリ名（シャープの後ろにスペースを入れてカテゴリを定義）
- [リンクタイトル](URL) {icon: アイコン名} {img: 画像パス} : 備考（ここにメモを記述）

# 💻 開発ツール（複数カテゴリを作ると自動でタブ化されます）
- [GitHub](https://github.com) {icon: github} : コード管理プラットフォーム。
- [Google](https://google.com) {icon: google} {img: https://images.unsplash.com/photo-1579546929518-9e396f3cc809?w=300} : 綺麗なロゴ写真をタイル画像に指定した例。
- [ローカルサイト](http://localhost:8080) {img: ./my-icon.png} : 同一フォルダ内にあるローカル画像を指定する例。
```
